# X Pre-Flight Gate (run before posting)

Run end to end at Step 7 and write the result into `CHECKLIST.md`. Several items are mechanical.

> If a single box can't be honestly ticked, don't post yet. Fix it, then re-run.

## The post

- [ ] **Hook tweet stops the scroll** — leads with the benefit / a specific or contrarian claim, not a
      category label or a contentless "a thread 🧵". (≤ 280 chars, counted.)
- [ ] **No external link in the main tweet** — the launch URL is in the **first self-reply** only.
- [ ] **Native media** attached to the hook (a 10–30s demo clip or a sharp screenshot/GIF), and a workflow
      clip/screens on the demo tweet.
- [ ] Thread follows the arc (hook → problem → what it is → demo → different → [proof] → CTA) and is tight —
      every tweet earns its place.
- [ ] **0–2 hashtags** total (usually none).
- [ ] The first-reply link is **UTM-tagged** (`utm_source=x`, or `twitter` — match whatever
      `launch-analytics` chose) and matches `analytics.md`.

## Compliance (rule line)

- [ ] **Bait scan passes** — zero "like & RT to win", "reply YES for the link", follow-for-follow, or any
      incentive tied to a like/RT/follow (see `antipatterns-compliance.md`).
- [ ] No bought engagement, engagement pods, follow-churn, or bulk/automated DMs/replies/likes/follows.
- [ ] Any engagement ask is to your **own** audience and genuine ("a repost would mean a lot"), not coerced.

## Account & timing

- [ ] Account tier checked — **Premium/Premium+** if reach matters (regular accounts trend toward ~0%
      median reach in 2026; third-party data).
- [ ] Post **time** set for when your most-engaged followers are online (a weekday-morning window is a
      common default).
- [ ] You (or a teammate) are **available for the first 30–60 minutes** to reply to every comment live
      (replies are the top positive signal; author replies are weighted heavily).
- [ ] Plan to **pin** the thread to the profile right after posting.

## Expectations (honesty check)

- [ ] A **realistic reach target** for this account's size — the durable wins are replies/conversation,
      profile + link clicks, and new followers, not the like count. No promise of virality.

## Scored self-critique (calibrate — don't rubber-stamp)

Score 0–4 each (4 = genuinely excellent; most real work is 2–3): **hook strength · thread tightness ·
demo media · link handling (in reply) · compliance (bait-free) · golden-hour readiness**.

- [ ] Total recorded (max 24). Any axis **≤ 2 → name the fix, apply it, re-score.**
- [ ] Revise-and-justify: for each change at this gate, one line on WHAT changed and WHY. Don't post with an
      un-actioned axis ≤ 2 or any compliance box unticked.
