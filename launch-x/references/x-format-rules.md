# X Launch Format Rules

Load at Step 3. The copy/skeleton lives in `thread-copy.md`; this is the **format, limits, media, and the
link/pin mechanics.**

## Limits

| | Value |
|--|--|
| Standard post | **280 characters** (free) |
| Premium "long post" | ~**25,000 chars** (collapses to ~280 + "Show more" in timeline) |
| Premium+ Articles | ~100,000 chars, rich text, Articles tab |
| Formatting | bold/italic on Premium tiers |

**Thread vs single post:** a **thread is the standard launch vehicle** (each reply/dwell/"show this
thread" feeds ranking and the hook stays a reply-magnet). A single Premium **long post** is an option for
one dense narrative — its first ~280 chars are the in-timeline preview, so make them the hook.

## The link rule (non-negotiable)

**Never put the launch URL in the main tweet** — X down-ranks link-in-body posts (see `x-mechanics.md`).
Put the product link in the **first self-reply** to the thread; replies aren't suppressed the way parent
posts are, so the message everyone sees keeps full reach while clicks still route. (Musk's nuance: a link
with a strong description/image gets *some* distribution, and X is testing an in-app browser to soften the
penalty — but **link-in-reply is the safe default** in 2026.)

## Hashtags

The Jan-2026 Grok ranker reads posts **semantically**, so hashtags are near-useless for discovery and
**>2 can trip a spam classifier** (a ~40% reach cut, third-party estimate). Use **0–2 max**, only if
genuinely relevant — usually none.

## Native media (non-negotiable)

The ranker favors **rich media**; X is **video-first**. **Native video uploaded to X outperforms a linked
YouTube/external video** (which also eats the link penalty). Lead the hook tweet with a **10–30s product
demo clip** or a sharp before/after **screenshot/GIF**; show the core workflow end-to-end on a later tweet.
Caveat: well-crafted native **text** can out-engage video in some samples — so aim for **media-rich AND
tightly written**, not media as a crutch.

## The hook tweet decides everything

Reportedly **~94% of threads get under 10 reposts**, mostly because the first tweet doesn't stop the scroll
in ~3 seconds. Strong 2026 hooks are **specific and personal** (a real result/timeframe/mistake), a **bold
or contrarian** claim, or an **open loop** — and lead with the **benefit, not the category**. Never open
with a contentless "a thread 🧵👇".

## The launch-thread arc

`HOOK (+ demo media)` → `PROBLEM (agitate)` → `WHAT IT IS (plain)` → `DEMO (show it working)` →
`HOW IT'S DIFFERENT (the wedge)` → `SOCIAL PROOF (optional)` → `CTA` — and the **link goes in the first
reply**, not the CTA tweet body. Full skeleton in `thread-copy.md`.

## After posting

1. **Pin the thread:** open tweet 1 → three-dot menu → **"Pin to your profile"** (keeps the launch at the
   top of the profile for the campaign).
2. **Protect the first 30–60 minutes:** reply to every comment live (author replies are weighted heavily —
   see `x-mechanics.md` and `timing-engagement.md`).
3. Post **when your most-engaged audience is online** (a weekday-morning window is a common default).
