# X Launch Copy (the X asset profile)

This is the **X asset profile** the orchestrator hands to the `launch-copy` skill — assets, voice, and the
thread skeleton — **and the inline fallback** if `launch-copy` isn't installed. Load at Step 3. Bind every
line to `.ulpi/launch/positioning.md`; honor the limits in `x-format-rules.md`.

**Profile to pass to `launch-copy`:** `channel` = x; `voice` = authentic, punchy, specific, no hype/no
corporate-speak (sound like a builder, not a brand); `compliance` = no external link in the main tweet
(link in the first reply), ≤2 hashtags, no engagement-bait; `assets` = hook tweet (≤280, 3–4 options),
thread body (the arc below), the first-reply link line. The skeleton below is the fallback.

## The hook tweet (≤280 — generate 3–4 options)

Lead with the **benefit or a specific/contrarian claim**, not the category. Patterns: a real
result/timeframe ("We cut X from 2 hours to 2 minutes"), an open loop ("We almost killed this product
twice. Today it's live:"), or a bold claim. Attach demo media. No link, no hashtags in the hook.

## Paste-ready launch-thread skeleton (link-in-reply)

```
TWEET 1 — HOOK (must stop the scroll; attach a 10–30s demo clip or sharp screenshot/GIF)
We just shipped [PRODUCT].
It [the one outcome people care about] in [timeframe] — without [the painful old way].
Here's what it is and why we built it 🧵
[ATTACH DEMO VIDEO / GIF / SCREENSHOT]

TWEET 2 — PROBLEM (agitate)
Every [target user] hits the same wall:
• [pain 1 — concrete]
• [pain 2 — concrete]
• [the cost: time/money/sanity]
We lived this for [N months]. Nothing solved it cleanly.

TWEET 3 — WHAT IT IS (plain language a stranger gets)
So we built [PRODUCT]: [one sentence, plain].
Think "[familiar thing] but [your twist]."

TWEET 4 — DEMO (show it working; attach a 10–15s workflow clip/screenshots)
Here's it in action 👇
[1-line caption of what they're watching]

TWEET 5 — HOW IT'S DIFFERENT (the wedge)
Why not just use [obvious alternative]?
• [differentiator 1 — checkable]
• [differentiator 2]
• [the one thing only you do]

TWEET 6 — SOCIAL PROOF (optional; drop if none)
[early result / metric / waitlist number / notable user]
"[short testimonial]" — [name/role]

TWEET 7 — CTA (still NO link in the body)
That's [PRODUCT]. If [target user], it's for you. We're live today.
👇 Link in the first reply. Reply with questions — I'm here all day.

REPLY 1 (self-reply to tweet 1) — THE LINK GOES HERE
Try it / get early access: [PRODUCT_URL]   ← UTM-tagged
(Link's here so the thread keeps full reach.)
```

## Single long-post variant (Premium only)

Use only with Premium. The first ~280 chars are the timeline preview — **make them the hook**. Use bold
sparingly for scannable section heads (The problem / What it is / How it works / Why it's different / Who
it's for / CTA). The **link still goes in the first reply**, not the body.

## Voice & slop bans

Authentic builder voice. Ban corporate-speak ("We're thrilled to announce…", "revolutionary",
"game-changing", "seamless"), buzzword soup, and fake precision. Lead every tweet with substance; make the
hook earn the thread. (See `launch-copy`'s `copy-craft.md` for the general craft.)
