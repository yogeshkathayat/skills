# LinkedIn Pre-Flight Gate (run before posting)

Run end to end at Step 7 and write the result into `CHECKLIST.md`. Several items are mechanical.

> If a single box can't be honestly ticked, don't post yet. Fix it, then re-run.

## The post

- [ ] **Hook earns the "…see more"** — leads with a felt problem / sharp number / personal-story opener
      (product named later), and survives the **~140-char mobile fold** (counted). No "I'm excited to
      announce…".
- [ ] Body **delivers value without requiring a click**; formatted with whitespace / one idea per line for
      dwell; ~1,300–2,000 chars.
- [ ] **No link-preview card in the body** — the launch link is in the **first comment** (UTM-tagged), or a
      plain-text URL near the end.
- [ ] **Format chosen for dwell** — document/carousel, a strong image, or native video (not a link card).
- [ ] Ends with **one genuine question** (a real discussion prompt — **not** "Comment YES" bait).
- [ ] **3–5** relevant, mostly-niche hashtags (no `#like/#follow/#comment`, no 10+).
- [ ] Posting from the **founder's personal profile** (company page re-shares later); 1–3 relevant tags.

## Compliance (rule line)

- [ ] **Bait/pod scan passes** — zero "Comment YES", "Tag 3 people", reaction-polls-for-reach, or
      `#like/#follow/#comment` (officially spam); **no engagement pod** or pre-arranged reciprocal swap; no
      comment-automation tools (see `antipatterns-compliance.md`).
- [ ] Any team/network ask is **genuine and relevance-framed** (their own words; the "Notify Employees"
      tool) — never scripted, never reciprocal-by-agreement.
- [ ] Tone is **~80/20 value-to-promo**, not a sales pitch.

## Behaviour & timing

- [ ] **Post is proofed** — you won't need to edit in the first ~60–90 min (edits in the momentum window
      lower reach; the first comment carries the link, so no edit needed).
- [ ] Post **time** set for a weekday slot your audience is online (Tue–Thu; test AM vs PM).
- [ ] You're **available for the first 60–90 minutes** to reply substantively to every comment (the ~30%
      lever); the **first comment with the link** is ready to paste on publish.
- [ ] Not posting **2+ times** the same day.

## Expectations (honesty check)

- [ ] A **realistic reach target** for this account's network — the durable wins are substantive comments/
      conversation, profile visits, and qualified followers, not the reaction count. No promise of virality.

## Scored self-critique (calibrate — don't rubber-stamp)

Score 0–4 each (4 = genuinely excellent; most real work is 2–3): **hook (earns the expand) · dwell-worthy
body · format choice · link handling · compliance (bait/pod-free) · golden-hour readiness**.

- [ ] Total recorded (max 24). Any axis **≤ 2 → name the fix, apply it, re-score.**
- [ ] Revise-and-justify: for each change at this gate, one line on WHAT changed and WHY. Don't post with an
      un-actioned axis ≤ 2 or any compliance box unticked.
